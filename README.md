# shapeshift.com code-exercise

---

## Endpoints:

### Look up exchange rate for given coin-type:
#### GET `/exchange-rate/:coin-a/:coin-b/:amount`

---

## Requirements:

- Node v7 or newer

	`nvm install v7`
	
	`nvm use v7`

## Setup:

`npm install`

## Running:

`npm start`

Server runs on **localhost:4000**

## Development:

`npm run dev`

## Building Production Bundle:

`npm run build`
